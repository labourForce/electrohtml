To use this script you should install PuTTY. See more at http://www.putty.org/

If you are in external network, go to start_deploy.sh, comment line #2 and uncomment line #3.
If you have local IP, line #2 should be uncommented and line #3 should be commented.

On Linux:
1. open terminal on electrohtml/deploy/
2. execute sudo sh start_deploy.sh

On Windows:
1. open bash emulator on electrohtml/deploy/
2. execute sudo sh start_deploy.sh

Within the server:
1. go to /home/usr/Documents/deploy/
2. execute sudo sh deploy.sh
